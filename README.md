## 프로젝트 소개
'EZ TOUR'는 모든 연령대가 '손쉽게' 여행을 할 수 있게 직관적인 워딩과 편의를 고려한 UI/UX 위주로
유저 중심의 여행 플랫폼 웹사이트입니다.
